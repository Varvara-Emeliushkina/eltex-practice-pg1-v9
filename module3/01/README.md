# Задание 01 (Процессы).
Написать программу, которая принимает аргументы запуска и обрабатывает их по следующим правилам:

1. Eсли аргумент является целым или вещественным числом, то выводится это значение и значение, умноженное на 2;
2. Eсли аргумент не является числом, то он выводится в исходном виде. Обработку делают родительский и дочерний процессы, разделяя задачи примерно поровну;  

## Запуск
```c
make
```
## Тест 
```c
./process 12 fgxss67s 22 -12
```
